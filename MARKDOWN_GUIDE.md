# Markdown Formatting Guide

## Headings
To create headings, use the `#` symbol followed by a space:

# Heading 1
## Heading 2
### Heading 3

## Bold Text
To make text bold, wrap it in double asterisks or double underscores:

**This is bold text**  
__This is also bold text__

## Italic Text
For italic text, wrap it in single asterisks or single underscores:

*This is italic text*  
_This is also italic text_

## Code Blocks
Use backticks for inline code and triple backticks for code blocks:

Inline code: `code`

```
This is a code block
You can write multiple lines of code here.
```

## Lists
### Unordered Lists
Use asterisks, plus signs, or hyphens:
- Item 1
- Item 2
  - Subitem 2.1

### Ordered Lists
Simply number the items:
1. First item
2. Second item
3. Third item

## Tables
Use pipes and hyphens to create tables:

| Header 1 | Header 2 |
|----------|----------|
| Row 1   | Data 1   |
| Row 2   | Data 2   |

## Links
To create a link, wrap the link text in brackets and the URL in parentheses:

[Google](https://www.google.com)

## Images
To add an image, use an exclamation mark followed by the alt text in brackets and the URL in parentheses:

![Alt text](https://example.com/image.jpg)
