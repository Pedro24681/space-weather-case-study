# Project Notebooks

This directory contains all Jupyter notebooks for each phase of the space weather case study:
- `01-data-load-and-eda.ipynb`: Exploratory data analysis
- `02-data-integration.ipynb`: Data integration and merging
- `03-correlation-and-events.ipynb`: Event detection and storm cataloging
- `04-impact-assessment.ipynb`: Final impact analysis and synthesis

Each notebook contains markdown commentary, code and visualizations.
